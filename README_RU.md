**Daltoon Bot** — это современное решение для управления VPN-сервисами в реальном времени через Telegram. Проект предоставляет чистый и автоматизированный интерфейс для развертывания, настройки и мониторинга широкого спектра VPN-протоколов на нескольких панелях.

Созданный как универсальная система, Daltoon Bot предлагает расширенную поддержку панелей управления, повышенную стабильность и современный веб-интерфейс для администрирования.

> [!IMPORTANT]
> Этот проект предназначен исключительно для личного использования. Пожалуйста, не используйте его в незаконных целях или в производственной среде.

## 🚀 Возможности

- **Поддержка нескольких панелей** — интеграция и синхронизация с панелями **Sanaei (3x-ui)**, **Reebeka** и **Pasarguard**.
- **Система кошелька** — встроенный баланс для пополнения счета и мгновенной покупки подписок пользователями.
- **Мониторинг в реальном времени** — автоматическое отслеживание трафика, лимиты использования для каждого клиента и статус активных подключений.
- **Панель администратора** — современный веб-интерфейс на React для полного управления пользователями, серверами и настройками системы.
- **Управление несколькими серверами** — контроль и масштабирование нескольких VPN-серверов из одного центрального бота.
- **Автоматическая установка** — скрипт быстрого запуска, который настраивает все зависимости, включая Python, Node.js и PM2.
- **Безопасность** — централизованная база данных SQLite с безопасной обработкой API и зашифрованной конфигурацией.
- **Управление процессами** — полная поддержка PM2 для высокой доступности и автозапуска служб в случае сбоя.

## ⚡ Быстрый запуск

Для мгновенной установки Daltoon Bot на ваш сервер Linux выполните команду:

```bash
bash <(curl -Ls https://raw.githubusercontent.com/mdaltoon10/Daltoon-Bot/main/install.sh)
```

## 🛠 Панель управления (`daltoon-dashboard`)

После установки вы можете управлять службами с помощью команды `daltoon-dashboard`. Это интерактивная консольная утилита (CLI), которая упрощает администрирование.

### 📋 Таблица команд

| Команда | Описание | Функционал |
| :--- | :--- | :--- |
| **[1] Update** | `🔄 Update Project` | Скачивание обновлений из GitHub и пересборка проекта. |
| **[2] Uninstall** | `🗑️ Uninstall` | Полное удаление проекта и всех фоновых служб. |
| **[3] Status** | `📊 View Status` | Просмотр потребления ресурсов CPU/RAM панели и бота. |
| **[4] Start** | `🚀 Start Services` | Запуск панели и бота в фоне с помощью PM2. |
| **[5] Stop** | `🛑 Stop Services` | Остановка всех запущенных служб Daltoon. |
| **[6] Restart** | `♻️ Restart Services` | Перезапуск всех служб (панели и бота). |
| **[7] Credentials** | `🔑 Change Login` | Изменение логина и пароля администратора для входа в панель. |
| **[8] Port** | `🔌 Change Port` | Изменение веб-порта панели управления. |
| **[9] Exit** | `🚪 Exit` | Выход из панели управления. |

## 🛠 Ручная установка

Выполните следующие шаги для ручной установки и запуска бота:

1. **Клонирование репозитория**:
   ```bash
   git clone https://github.com/mdaltoon10/Daltoon-Bot.git
   ```

2. **Запуск установщика**:
   ```bash
   cd Daltoon-Bot
   chmod +x install.sh
   ./install.sh
   ```
